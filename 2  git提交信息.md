# git提交信息
## 提交信息格式
```
<type>(<scope>): <subject>
// 空一行
<body>
// 空一行
<footer>
```
### type 参考
![上传信息](https://i.imgur.com/Rs6r9HU.png)
### scope
影响的范围
### subject
简短的描述
### body
具体的描述
多行描述用`- 内容`格式
如下:
```
- 改动1：内容
- 改动2: 内容
```
### footer
不经常用到
